1. História dos computadores

A palavra “computador” vem do verbo “computar” que, por sua vez, significa “calcular”. Sendo assim, podemos pensar que a criação de computadores começa na idade antiga, já que a relação de contar já intrigava os homens. Dessa forma, uma das primeiras máquinas de computar foi o “ábaco”, instrumento mecânico de origem chinesa criado no século V a.C. Assim, ele é considerado o “primeiro computador”, uma espécie de calculadora que realizava operações algébricas.

Máquinas de Contagem (Antiguidade): O conceito de contar e calcular remonta à antiguidade. Ábacos e outros dispositivos mecânicos simples foram usados por várias civilizações para realizar cálculos básicos.

Máquina Analítica de Charles Babbage (século XIX): Considerado o "pai do computador", Charles Babbage projetou a Máquina Analítica no século XIX. Embora nunca tenha sido construída em sua totalidade durante sua vida, o projeto de Babbage incluía elementos fundamentais de um computador moderno, como a unidade de controle, a unidade aritmética e a memória.

Primeiros Computadores Eletromecânicos (início do século XX): Durante a Primeira Guerra Mundial, máquinas como a "Mark I" de Howard Aiken e a "ENIAC" (Electronic Numerical Integrator and Computer), construída nos anos 1940, foram alguns dos primeiros computadores eletromecânicos.

Era dos Transistores (década de 1950): A invenção do transistor na década de 1940 permitiu a miniaturização dos componentes eletrônicos e a criação de computadores mais eficientes. Isso marcou o início da era dos computadores de segunda geração.

Desenvolvimento dos Circuitos Integrados (década de 1960): A criação dos circuitos integrados permitiu a integração de vários transistores em um único chip, resultando em computadores menores, mais rápidos e mais confiáveis.

Era dos Microprocessadores (década de 1970): O desenvolvimento do microprocessador, como o Intel 4004, marcou o surgimento dos computadores pessoais. Empresas como Apple e Microsoft desempenharam papéis significativos nessa fase.

Computação Pessoal e a Internet (décadas de 1980 e 1990): O aumento da capacidade de processamento, a popularização dos computadores pessoais e a expansão da Internet transformaram a sociedade e a forma como as pessoas interagem com a informação.

Era Contemporânea (século XXI): Avanços contínuos em hardware, software, inteligência artificial, aprendizado de máquina e computação em nuvem têm definido a era contemporânea dos computadores. Tecnologias como smartphones, tablets e dispositivos vestíveis também se tornaram onipresentes.

2. O que são clientes e servidores?

Um servidor é um recurso dentro de um sistema computacional, capaz de processar aplicações, armazenar dados e prestar serviços, enquanto o cliente é um computador que solicita esses serviços e recursos. Em uma infraestrutura de TI, ambos formam a arquitetura conhecida como “cliente-servidor”. Uma estrutura cliente-servidor é um modelo de arquitetura de rede em que um programa de computador (cliente), solicita um serviço ou recurso de outro programa de computador (servidor), recebendo uma resposta com as informações solicitadas por meio de protocolos de rede em infraestruturas de TI.

A comunicação cliente-servidor é baseada em troca de mensagens, onde o cliente envia requisições e o servidor responde com as informações requisitadas. Este tipo de modelo é amplamente utilizado em rede de computadores, onde vários dispositivos podem se conectar a um servidor central para acessar recursos compartilhados, como arquivos, impressoras, bancos de dados, aplicativos e serviços de internet.

A arquitetura cliente-servidor pode ser implementada de diferentes maneiras, como em aplicações web, jogos online, sistemas de gerenciamento de bancos de dados, entre outros.

3. Como é a estrutura básica de um HTML?

A estrutura básica de uma página HTML, pode ser dividida em 4 tags principais: doctype, HTML, head e body.

  • Doctype: instrução usada para informar ao navegador que o documento em questão está usando a versão HTML5. Isso permite o uso de todos os atributos disponíveis para essa versão.

  • HTML: essa tag informa ao navegador que o documento está seguindo a estruturação de um documento HTML5.

  • Head: tag utilizada para definir informações de cabeçalho, como idioma, título e metadados. Além disso, essa tag também é utilizada para inserir os links para arquivos CSS e Javascript. Por fim, essa tag é localizada dentro da tag HTML

  • Body: essa tag é responsável por definir o corpo do documento. Nessa tag serão adicionados todos os elementos visíveis da página, como textos, imagens, títulos, vídeos, formulários, entre outros.

Dica: Para adicionar todas essas tags de maneira simples no VScode, usa-se o sinal de exclamação.

4. Listas ordenadas e não ordenadas

Existem dois tipos de listas que podem ser utilizadas em um documento HTML: listas ordenadas e listas não ordenadas.

As listas ordenadas <ol> são utilizadas quando se quer montar uma lista na qual a ordem dos elementos importa.

Já as listas não ordenadas <ul> são utilizada quando se deseja montar uma lista na qual a ordem dos elementos não importa.

Estrutura de uma lista:

tag <ol> ou <ul>, para estabelecer a estrutura básica, seguido da tag <li> para cada elemento que se deseja adicionar na lista.

5. Outras tags

  • <i>: usada para deixar os textos em itálico.

  • <blockquote>: usada para inserir citações de tamanho longo.

  • <small>: usada para tornar um texto em um tamanho menor.

  • <sub>: serve para adicionar um texto subescrito no documento. Ex: O2

  • <sup>: serve para definir um texto subroescrito no documento. Ex: 22

  • <font>: serve para definir o tipo de fonte no texto (não é mais utilizada).

  • <abbr>: serve para definir um texto abreviado. Ex: HTML (coloque o mouse por cima para ver)

  • <del>: serve para definir um texto deletado. Ex: texto deletado
